RHINO

Rhino was the first vehicle fabricated by the team in the year 2014. It was the first fully running prototype.






Design

Width - 1243.2mm
Length - 1588.0mm
Wheelbase - 1080mm
Trackwidth - 1040mm
Weight - 250kg

Engine

Discover 135,
Power - 13BHP @ 8500rpm
Torque - 11.9Nm @6500rpm

Wheels

Front wheel - 20x8-12
Rear wheel - 20x12-12

Transmission

Type - Single reduction transmission

Suspension Type

Front - Double wishbone
Rear - Trailing arm

Brakes

Type - Hydraulic disc brakes front and rear

Steering

Type - 4 bar linkage centralised point
